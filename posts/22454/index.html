How to install docker on Mac M1 series?

在本範例你會學到：

  • 如何於 新版的 mac m1 系列中安裝 docker

在本範例你需要先準備好：

  • 一台稍微便宜一點的 M1 晶片 mac

Mac系列新機，均搭載蘋果專為Mac設計的首款晶片M1，以Arm架構設計Apple Silicon處理器的晶片M1，可以擁有更好的效能體驗，不過一開始卻有許多的災情尤其是很多程式的相容性，在安裝開發環境的時候很多人應該都撞了不少牆，今天就來簡單介紹大家常用的 docker 安裝吧！

1.安裝 Rosetta 2

這個東西老實說我不太知道是幹嘛的，還沒特別研究，有空閒我會再補充，不過在安裝其他程式的時候也會遇到他(ex:下一篇會介紹的 python install pandas 問題)。而在 docker 的官網上也已經列出了相關的 issue 給大家參考，就照著範例安裝吧：

softwareupdate --install-rosetta

2.下載 docker dmg

這邊有釋出了一個 preview 版本(官網下載點 Apple M1 tech preview build)，請各位自行下載。

3.啟動

一切就緒後，點兩下啟動 docker 鯨魚圖案，正常的依照正常程序安裝就可以了！過程中可能會問一些警告性的問題，都按確定或是ok就可以了！
